## Installation

```bash
$ npm install

$ docker-compose up -d
```

## Running the app

```bash
# development
$ npm run start:local

# homolog
$ npm run start:homolog

# production mode
$ npm run start:prod
```

## Test

```bash
# e2e tests
$ npm run test:e2e

# test on push
$ npm run test:push:e2e
```
